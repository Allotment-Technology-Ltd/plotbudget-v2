# Contributing to PLOT

Thanks for your interest in PLOT! We're a small team in beta and not actively accepting external PRs at this stage.

## For Beta Testers

Found a bug? Have feedback?

1. **Open an Issue** — Use the bug report template
2. **Include details** — Steps to reproduce, OS, browser, app version
3. **Be respectful** — We're all here to make PLOT better

## Code Quality

This codebase is built to production standards:
- TypeScript strict mode
- OWASP-compliant security practices
- WCAG 2.2 AA accessibility
- Comprehensive test coverage

We use AI (Claude + Cursor) for implementation, but all code is reviewed by humans and held to high standards.

## Reporting Security Issues

Found a vulnerability? Email **security@plotbudget.com** instead of opening a public issue.

See [SECURITY.md](./SECURITY.md) for details.
